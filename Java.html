Example Programs

  • Program that calculates student averages, course averages, class median, highest and lowest grades, and awards a scholarship to eligible students The program runs until the user has entered as many classes and students desired and prints the results in a tabular format


  • Program that creates and manipulates a custom stack class Class Stack pushes and pops items onto and off of the stack, as well as telling the user when the stack is full or empty. Class Stack also enables the user to push many items onto the stack at one time


  • Program that creates and manipulates a custom linked list class The Link class allows the user to add items, delete items, sort those items in ascending and descending order, shuffle the items, and allows the user to delete items from anywhere in the list Program also utilizes the JOptionPane GUI class


  • Program that creates and manipulates a custom priority queue class The main program instantiates two priority queue objects, populates them with data, prints each queue's data, and calculates the Union, Difference, and Intersection of the priority queues, then prints the information in tabulated format


  • Program that uses inheritance to instantiate objects for salary workers and hourly workers. The SalaryWorkers class inherits the ID field, and set and get ID functions from the Employees class. One of the PrintPay methods in the SalaryWorkers class overrides the PrintPay of the super class by using the @override tag while the other uses a different parameter list
