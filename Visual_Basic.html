This is an Access database application that I wrote for the Welding Department of Navarro College where I attended classes. Various aspects of the application are shown as screenshots. The VisualBasic code in this example allows for functionality such as cascading dropdown boxes, in which certain selections call for different dropdown choices, query selections using wild cards to search for names or other identifiers in the database to be quickly accessed with only a few keystrokes, and reports to show the results.

  • Access application for Navarro College's welding department


  • Wild cards in the code allow the user to look for students by their first name, last name, or anything close to give a list of possible students to choose from. The user can then view or manipulate that student's information


  • Drop down boxes for level, section, and practicals provide the only available options to reduce entering incorrect information. Each level and section have different sets of practicals depending on the selection.


  • Forms for adding test scores, practical scores, students, and student information


  • Reports allow the user to search for and view a student's practical and test scores


  • Other reports allow the user to view a list of students and grades for particular levels or sections
